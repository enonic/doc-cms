= Long input type

:imagesdir: images
:toc: right

A simple input for storing integer numbers.

== Appearance

image::Long.png[Number input showing a large number, 700]

== Usage

[source,yaml]
----
- type: "Long"
  name: "my_long"
  label: "My Long"
  default: 42  # <1>

- type: "Long"
  name: "degrees"
  label: "Degrees"
  min: 0  # <2>
  max: 360  # <3>
----
<1> *default* can be any Integer number
<2> *min* optionally specifies the minimum allowed value
<3> *max* optionally specifies the maximum allowed value

== Output

*Value type:* `Long` — not included in `_alltext`

[source,JSON]
----
{
  "my_long": 42
}
----

.Multiple occurrences
[source,JSON]
----
{
  "my_long": [
    42,
    99
  ]
}
----
