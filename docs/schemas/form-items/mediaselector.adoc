= MediaSelector input type

:imagesdir: images
:toc: right

The MediaSelector is a specialized version of the <<contentselector#, ContentSelector>> that is limited to selecting media only (content of type "media:*"), including also uploading new media (any kind of files).

== Appearance

image::Media-selector.png[Media selector with a grid of media items, 500]

== Usage

[source,yaml]
----
- type: "MediaSelector"
  name: "my_media"
  label: "My Media"
  occurrences:
    min: 0
    max: 1
  allowContentType:  # <1>
    - "media:archive"
  allowPath:  # <2>
    - "${site}/*"
  treeMode: true
  hideToggleIcon: true
----
<1> MediaSelector supports the same configuration options as <<contentselector#, ContentSelector>>, but is limited to using `media:*` content types in the `allowContentType` configuration.
<2> Just like <<imageselector#, ImageSelector>>, MediaSelector by default displays all media items starting from the content root.

== Output

*Value type:* `Reference` — not included in `_alltext`

The stored value is the content ID of the selected media item. Like <<contentselector#, ContentSelector>>, the value is a reference to the media content item.

[source,JSON]
----
{
  "my_media": "f3076b5c-ea45-4c8b-8c06-1f87b8d8cdd9"
}
----

.Multiple occurrences
[source,JSON]
----
{
  "my_media": [
    "f3076b5c-ea45-4c8b-8c06-1f87b8d8cdd9",
    "a1b2c3d4-e5f6-7890-abcd-ef1234567890"
  ]
}
----
